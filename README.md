# Droguería Backend

Proyecto estudiantil desarrollado como parte del programa **Tecnólogo en Análisis y Desarrollo de Software (SENA)**.  
Este backend gestiona la información básica de una droguería, permitiendo operaciones sobre productos, usuarios y ventas.

## 🚀 Tecnologías utilizadas
- **Node.js** – entorno de ejecución
- **Express.js** – framework para crear la API REST
- **JavaScript (ES6+)** – lenguaje principal
- **dotenv** – manejo de variables de entorno
- **Git/GitHub** – control de versiones
